User Manual

v1.006

1. Introduction

The Earthwork Calculator is a specialized tool for calculating Cut/Fill volumes using the Cross Section Method. It supports complex scenarios such as:

  • Variable KM Lengths (e.g., Km 506 having 1856m).
  • Specialized Work Types (Cess Repair, Cutting Widening).
  • Bridge/Volume Exclusion.
  • Excel Integration for bulk data processing.

2. Getting Started

  1. Launch: Open EarthworkCalculator.exe.
  2. Project Details: Fill in the project info (Zone, Division, Line, etc.) at the top. These appear in the PDF Report headers.
  3. Configuration:
    • Work Type: Select the mode of calculation (see Section 5).
    • Offsets: Define your cross-section points (e.g., -10, -5, 0, 5, 10 meters from center).

3. Data Input Methods

Method A: Excel Import (Recommended)

This is the fastest way to process data.

  1. Download Template: Click the Template button to get Earthwork_Template.xlsx.
  2. Fill Data:
    • Chainage: Enter as decimal (e.g., 506.400) OR formatting is handled automatically.
    • Offset: Distance from center (negative for Left, positive for Right).
    • Levels: Enter Initial_Level and Final_Level. If using "Standard Levels", fill Standard_Level.
    • KmLength (Critical): In case of short/long Kms, actual length of long/short Km to be entered in this cell against the last chainage for that Km. Otherwise keep this blank. For eg, if Km 506 is having only 995m length and from there km 507 starts. Last chainage where level taken in km 506 is 506/990 and next level is at 507/020. In this situation type 995 in the KmLength column against Km 506/990. System will automatically assume distance between 506/990 and 507/020 as 25m for volume calculation.
    • Remark: In case of any deduction between any 2 chainages, for eg availability of bridge in this location and earthwork done is zero, type "Skip" or "Deduction due to Bridge 542" in the column against end chainage(highest chainage among the two), so system will give output earthwork volume between these two chainages as zero.
  3. Import: Click Import, select your file. The App will process and display the data.

Method B: Manual Entry

  1. Configure Grid: Enter Start Chainage, End Chainage, and Interval (in Meters).
  2. Generate: Click Generate Grid.
  3. Fill Levels: Manually type Initial/Final levels in the grid cells.

4. Understanding Chainage Formats

The app uses a Slash Format (Km/Meter) to handle complex chainages accurately.

Input Display Meaning
506.400 506/400 Km 506, Meter 400
506.040 506/040 Km 506, Meter 40
506.1950 506/1950 Km 506, Meter 1950 (Excess Length)

Note: If you enter 500.1100 in the "End Chainage" box, the app smartly recognizes this as 1100 meters (Excess), not 110 meters.

5. Work Types

  • Cutting Widening: Calculates Cut Volume only. Used for widening existing cuts.
  • Cess Repair: Calculates Fill Volume only. Used for repairing embankments.
  • Normal: Calculates both Cut and Fill.

6. Generating Reports

  1. Calculate: Click Calculate Volume. The results table will populate.
  2. Preview Graphs: Click View Graph on any row to see the cross-section visually.
  3. Export PDF: Click Export PDF. The report includes Project Summary, Detailed Tables (Mean Area x Distance), and Cross Section Graphs.

7. Troubleshooting

  • "Start > End" Error: Ensure you are entering valid chainages. For excess lengths (e.g., >1000m), ensure you type the full 4 digits (e.g., .1100).
  • Calculations look wrong? Check the Remark column. If it says "Bridge", volume is intentionally zero.
  • Reset: Use the Reset button to clear data. The app will retain your last selected "Work Type" for convenience.
